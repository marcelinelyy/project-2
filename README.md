## Парсер бизнес-кейсов с ads.vk.com
Простой парсер для сбора бизнес-кейсов с сайта ads.vk.com/cases. Выводит название кейса, дату его публикации и ссылку на сам кейс.

## Требования:
- Python 3.10+
- pip для установки зависимостей (beautifulsoup4, requests)

## Установка: 
pip install beautifulsoup4 requests

python parse_vk_cases.py

## Как работает код? 
Получение данных: Отправляет HTTP-запрос к ads.vk.com/cases

Парсинг HTML: Использует BeautifulSoup для анализа страницы

Фильтрация ссылок: Отбирает только ссылки, содержащие "/cases/"

Извлечение данных:
- Текст ссылки (содержит дату и название)
- Преобразование ссылки в абсолютный URL
- Обработка: Разделяет дату и заголовок, форматирует вывод
  
Вывод: Показывает все найденные кейсы в консоли
