##Парсер бизнес-кейсов с ads.vk.com
#vk_ads_cases.py скачивает HTML-страницу с кейсами на ads.vk.com, вытягивает из карточек данные о кейсах, нормализует данные и сохраняет их в JSON. По умолчанию используется первая страница кейсов (p=1).

##Требования:
- Python 3.10+
- pip для установки зависимостей (beautifulsoup4, requests)

##Установка: 
pip install beautifulsoup4 requests
python parse_vk_cases.py

##При запуске Вам предложат выбрать режим работы:
- Одна страница - парсит указанную страницу
- Несколько страниц - диапазон страниц (например, с 1 по 5)
- Все страницы - автоматически парсит страницы 1-18

##Как работает код? 
- Интерактивный выбор параметров парсинга (одна страница/диапазон/все страницы)
- HTML скачивается через requests (если не используется сохранённый файл)
- Страница разбирается BeautifulSoup: ищутся ссылки <a> содержащие /cases/ в атрибуте href
- Из каждой карточки вытаскиваются:
- Название кейса (из заголовков h1-h4 или значимого текста)
- Ссылка на кейс (преобразуется в абсолютный URL)
- Дата публикации (ищется по наличию цифр и названий месяцев)
- Дубликаты (одинаковое название и ссылка) отбрасываются
- Список кейсов записывается в JSON в UTF-8 с русскими символами, после чего печатается в stdout
